-CGPA Trend Analysis Using Python

This project analyzes and visualizes a student's semester-wise CGPA trend using Python, Pandas, and Matplotlib.
It provides insights into academic performance across the 1st to 5th semesters.

-Project Overview

This project includes:

-Data Loading

Reads data.csv containing semester CGPA values.

-Data Selection

Extracts only the columns:
1st, 2nd, 3rd, 4th, 5th

- Requirments

pandas
matplotlib

- installation
pip install pandas matplotlib

-Data Visualization

Creates the following graphs:

Bar Chart: CGPA Trend Across Semesters

Pie Chart: CGPA Contribution by Semester

These visualizations help understand performance patterns clearly.

-Dataset Format

Your data.csv should contain:

1st	2nd	3rd	4th	5th
8.5	8.7	9.0	8.9	9.2
